About SImTrad & The Developer

The Platform

SImTrad uses dynamic algorithms to simulate stock and cryptocurrency price action. Users can deposit virtual funds, build a portfolio, and track their performance via high-fidelity charting tools.

Built entirely using a Python backend and a responsive Bootstrap frontend, it serves as a lightweight, lightning-fast educational tool for practicing trading without the financial risk.
